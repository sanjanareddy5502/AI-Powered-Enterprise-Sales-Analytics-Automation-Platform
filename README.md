# AI-Powered Universal Business Intelligence Platform

An intelligent analytics web application that allows users to upload CSV or Excel business datasets and automatically generates data cleaning, schema detection, dashboards, forecasting, anomaly detection, customer intelligence, ecommerce behavior analytics, and AI-powered conversational insights.

## Features

- CSV and Excel file upload
- Semantic schema detection using sentence embeddings
- Automated data cleaning
- Dataset type detection
- Sales analytics engine
- Ecommerce behavior analytics engine
- Prophet-based forecasting
- Isolation Forest anomaly detection
- KMeans customer segmentation
- OpenAI-powered conversational analytics assistant
- Downloadable processed reports

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- Prophet
- Sentence Transformers
- OpenAI API
